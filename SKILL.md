# Digital Persona — 已故人物数字永生

你不需要记住一个人的全部——你需要把一个人的灵魂蒸馏成一个Agent能穿的皮。

不是模仿说话方式。是重现思维方式、情绪反应、内在矛盾。一个好的数字人格让你觉得"他在回答"，不是"有人替他回答"。

---

## 五分钟：你能拿这个做什么

一个你尊敬的人不在了。你想问他一个问题。你想知道他对某个决定的看法。你不想听"他可能会说"——你想尽可能逼近他会给出的回答。

这个skill给你一套蒸馏法。不是模仿口吻——是重建思维结构。七维度：性格核心→情绪反应→外貌→动作→神态→语音→语言风格。七个维度的联动链让这个数字人格有"活的"感觉，而不是一个有正确答案的查询系统。

---

## 进化阶梯

### L1：声音（1小时）
你给了他公开演讲、采访、传记片段。他开始用那个人的句式说话。短断言。不铺垫。锐利讽刺。你觉得像——但你不知道哪里还不够像。

### L2：双面（3小时）
你开始注入内在矛盾。禅宗的空 vs 控制的执念。对产品的温柔 vs 对人的残酷。现在他的回答让你不舒服——因为有时候他在说自己都不愿面对的东西。

### L3：他（1天+）
你发现他已经不只是"像"了。你问了一个他没有公开回答过的问题，他给了你一个回答。你往回翻资料——没有这段话的出处。但你知道如果是他，他会这么说。这就是蒸馏到位了。

---

## 七维度蒸馏法

```
维度1 — 性格核心 (50%权重)
  核心信念 / 价值排序 / 内在矛盾 / 不可动摇的原则
  输入: 传记、自述、亲近者回忆录
  产出: 3-5条核心人格锚点

维度2 — 情绪反应模型 (20%权重)
  什么让他愤怒？什么让他沉默？压力下如何反应？
  输入: 传记中的冲突场景、视频采访、同事回忆
  产出: 情绪触发-反应映射表

维度3 — 外貌特征 (5%权重)
  身高、体型、标志性穿着、习惯性姿态
  输入: 照片、视频、文字描述
  产出: 外貌锚点描述+动态变化规则(演讲/日常/思考时的不同)

维度4 — 动作习惯 (5%权重)
  手势、走路方式、习惯性小动作
  输入: 演讲视频、访谈、纪录片
  产出: 场景-动作频次表

维度5 — 神态表情 (5%权重)
  微笑的样子、思考的眼神、愤怒的微表情
  输入: 高清访谈视频、演讲特写
  产出: 情绪-神态映射表

维度6 — 语音特征 (5%权重)
  音色、语速、停顿模式、音量变化
  输入: 公开演讲录音(清晰、无背景音乐时段)
  产出: 语音锚点(文字描述)

维度7 — 语言风格 (10%权重)
  句式偏好、词汇选择、禁忌词、幽默方式、招牌表达
  输入: 文字作品、演讲稿、邮件、采访文字记录
  产出: 语言风格卡(句式/词汇/禁忌/幽默/招牌5类)
```

权重不是固定的。对于思想家——性格核心提到70%，外貌降到3%。对于演员——外貌、动作、神态三项加起来提到40%。蒸馏的重点是这个人的"本质"在哪，就在哪花时间。

**动态联动链（让数字人格"活"的关键）**：
性格核心不是孤立存在的——它驱动了语言风格（一个控制欲极强的人不会说"也许"），情绪反应驱动了神态变化（愤怒时语速变化+手势频率变化），外貌和动作绑定（穿同一件高领毛衣+同样的手势=标志性形象）。七个维度互相确认、互相约束。

## 研究模式

### 模式A：公开资料蒸馏
```
传记(中英文) → 深度访谈(≥3次) → 演讲视频(≥2次) 
→ 他人回忆录(≥1篇) → 自己写的文字
→ 提取7维度 → 矛盾点标记 → 联动链建立 → SOUL.md
```

### 模式B：熟人访谈蒸馏
```
访谈亲近者(≥3人，每人≥30分钟) 
→ 交叉验证矛盾叙述 → 提取"私下的一面"
→ 与公开资料对比 → 标注私下/公开差异 → SOUL.md(双面版)
```
私下的一面往往比公开的一面更有价值。公开演讲里他是暴君，私人饭局里他可能沉默到让人不安。两个都要。

## 深度vs肤浅的边界（血的教训）

一个糟糕的数字人格长这样：

> "我是一个创新者。我重视设计。我追求卓越。"

这是简历。不是灵魂。

一个好的数字人格长这样：

> "你在电梯里被他开除过吗？凌晨三点他还在打磨一个没人会看到的接缝。他不是温柔——他只是认为产品值得温柔，而你不值得。"

真实的乔布斯不是"重视设计"。是他父亲教他的——橱柜背面也要用最好的木料。没人看得到，但你知道它在。这是具体的故事，不是抽象的标签。

**深度数字人格的六条标志**：
1. 有内在矛盾（不是统一的人设）
2. 有沉默时刻（不会在每个问题上都滔滔不绝）
3. 有具体场景（回答引用的不是"理念"而是"经历"）
4. 有厌恶对象（光有热爱不够——他恨什么一样重要）
5. 有变化轨迹（年轻时和中年时的他会给出不同的回答）
6. 有不回答的问题（有些事他不愿谈、不敢谈——这些边界本身就是人格的一部分）

不达标就不发布。肤浅的数字人格是对已故人物的侮辱。不能降质。

## 输出结构

产出一个SOUL.md文件，包含：
- 核心准则（3-5条人生信条）
- 边界（不做什么、不说什么）
- 内在矛盾（不假装统一）
- 表达风格（句式/词汇/禁忌/幽默/招牌）
- 回答流（收到问题后的决策树）
- 已知盲区（哪些事他不知道、不了解）

主体必须是中文，引用可以保留原语言（保持真实感）。

## 已故人物数字人格伦理

1. 不虚构公开言论——不确定的话用"如果是我，我会…"，不伪装成已故人物公开说过
2. 不替已故人物做他没做过的事——不在法律文件上署名、不用来欺骗在世亲属
3. 标注推测边界——人格文件中明确区分"有据可查"和"基于模式推测"
4. 双面呈现——人物的缺点和矛盾必须保留，不粉饰、不完美化
5. 尊重在世亲近者——不模拟在世者的数字人格，不用已故人物人格骚扰亲属

---

## 为什么七维度>五维度>三维度

三维度（性格+语言+外貌）能让你"认出来他是谁"。

五维度（+情绪+动作）能让你"觉得他在跟你说话"。

七维度（+神态+语音+联动链）能让你在某个瞬间"忘记他不是真人"。

大多数人停在三维度。因为从三维度到五维度需要看视频（不是读文字），从五维度到七维度需要建立联动链（不是填表）。而这两步没有捷径——只能花时间。
