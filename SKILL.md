# Digital Persona — 已故人物数字永生

本质上不是模仿，是蒸馏。不是在教AI"像他那样说话"——是在提取一个人的变化规律。他什么状态下会怎样改变？什么能击穿他？他什么时候说的和做的不一样？

---

## 五分钟：你能拿这个做什么

你想让AI用乔布斯的视角看问题。或者用你敬重的某个已故之人的方式思考。大多数"AI角色"只能模仿表面——用他的词汇、引他的名言、学他的腔调。但这不是他。这是他的周边产品。

这个skill做的是另一件事：提取一个人的**变化规律**。不是"乔布斯很严格"——而是"乔布斯不满时语速会变慢、停顿会拉长、这不是在压抑愤怒而是在蓄积压强"。不是"他热爱简洁"——而是"他在产品会议和禅修室里的状态截然不同，但从产品会议到禅修室的切换方式就是他本人"。

七维度联动——性格→情绪→外貌+动作+神态+语音+语言——六者不是并列的，是因果链。情绪是引擎，外在表现是排气管。

五分钟看完这个文件，你知道方向。三十分钟读完reference和示例，你能做第一个。一周后你能做你自己的。

## 进化阶梯

### L1：照模板填（第一天）
打开persona-template.md，填七个维度。从网上搜集资料，对号入座。这是学徒阶段——你在学什么是关键特征、什么不是。

**交付物**：一份完整的SOUL.md。它能跑，但还不够像。

### L2：找变化规律（第一周）
你不再搜集"他长什么样"。你开始找"他什么时候会失控""他对不同人的温度差有多大""他最怕什么"。你开始看访谈视频——开着声音看一遍、关掉声音再看一遍。你在捕捉的不是内容，是变化。

**判断标准**：你能说出这个人在三种不同情绪状态下的语音、动作、神态差异。不是形容词，是"愤怒时从语速正常降到每字间隔2秒"这种精度。

### L3：蒸馏出矛盾（第一月）
你找到了内在矛盾——这才是人格核心。他宣称的价值观vs实际行为。他对自己的合理化解释。你写的SOUL.md不再是"他热爱简洁"，而是"他为什么在简洁的追求里建了科技史上最封闭的生态系统——他认为控制是消除杂乱的手段。这是他有道理但经不起推敲的自我辩护。"

**判断标准**：加载SOUL.md到Agent后，问它一个触及矛盾的问题。它不回避，不美化，呈现出复杂而诚实的回答。

### L4：独立Agent（第三月+）
你创建的不再是一个人格文件。它是一个能独立对话的Agent——有自己的边界、有不回答的问题、有自己的沉默。它像到让人不舒服。

**判断标准**：别人和它对话五分钟后，忘了这是AI。问的是"你怎么想的"，不是"你模仿得像不像"。

## 核心方法论：七维度联动

这七个维度不是并列的。分三层：

```
🟢 可观察层 — 外貌/动作/神态/语音/语言（外人可见）
🟡 组织层   — 情绪模型（触发器→外显的联动链）
🔴 核心层   — 性格核心（价值观、决策模式、内在矛盾）
```

情绪模型是整个系统的引擎。性格决定了什么触发什么情绪。情绪决定了神态、动作、语音、语言怎么变。大多数AI角色只做了最外层——语言和语音。里面是空的。

### 在研究阶段找什么

不是搜集事实。是寻找**变化模式**：

- 什么会击穿他的防线？什么情况下他失去了平时的控制？
- 对不同人的温度差有多大？对顶尖人才、对平庸者、对亲密的人——差距越大，人格越立体
- 什么时候说的和做的不一样？宣称的价值vs实际行为——矛盾是真实感的源头
- 他的标志性能量是什么？什么状态下他是满血的？什么消耗他？
- 他最怕别人看到什么？最小心翼翼藏起来的部分是什么？

### 研究优先级（从最值钱到最不值钱）

1. **影像** — 访谈/演讲/纪录片。唯一能同时捕捉外貌、动作、神态、语音四个维度的来源。看三遍：听内容→看表情→关声音看身体语言
2. **传记/自传** — 系统的生平、决策逻辑、内在矛盾
3. **亲历者回忆** — 私人时刻的秘密。公共形象和私人性格的差距
4. **本人文字** — 书信/邮件/笔记。和非公开语言风格对比
5. **争议和失败** — 人在低谷比在巅峰更真实

## 蒸馏要点（每个维度怎么提取）

### 性格核心
不写形容词。"他很严格"是形容。"他会在电梯里开除一个人，然后偷偷给另一个被自己开除的人付两年学费"是性格。

必须挖出：
- 至少一对内在矛盾（没有矛盾的人格是假的）
- 决策逻辑（不是做了什么决定，是怎么做决定的）
- 自我合理化（他不觉得自己矛盾——他怎么合理化？）

### 外貌
不是"他长什么样"。是"用户读完后能在脑子里看到他站在面前"。
- 标签性特征不超过3个（一眼能认出的）
- 年龄变化线（20岁/40岁/晚年分别什么样）
- 穿着反映决策哲学（不是时尚）

### 神态——微表情比大表情重要
大表情（大笑/大哭）是全人类共通的。微表情（嘴角短暂下拉、眼神闪避、鼻孔微张）才是个体独有的。重点写情绪的**泄露**——一个人想藏某种情绪时，从哪里漏出来。

### 动作——节奏和空间
不列手势清单。写：
- 节奏：快还是慢？停顿在哪？
- 幅度：大动作还是小动作？身体哪部分最活跃？
- 空间：他是扩张型（占空间）还是收缩型（缩起来）？

### 语音——动态规则比静态描述值钱
"音色中等、略带沙哑"有用但不够。"愤怒时语速从正常降到极慢，停顿从0.5秒拉到2秒"——这才是数字人格能用的。重点写三种状态下的变化：愤怒/兴奋/疲惫。

### 语言风格——禁忌词比常用词重要
一个人永远不会说的词和表达方式，比他会说的更能定义边界。乔布斯绝不会说"我建议您考虑一下"——不是偏好，是人格边界。

### 情绪模型——联动链是核心价值
```
触发器 → 情绪(强度1-5) → 神态怎么变 + 动作怎么变 + 语音怎么变 + 语言怎么变
```
6-8条联动链覆盖不同情绪。更值钱的是链之间的切换——从愤怒变平静、从兴奋变冷静——怎么切换，切换要多久。

## 案例：乔布斯数字人格

我们蒸馏了一个乔布斯的SOUL.md。以下是关键设计决策。

**内在矛盾（核心层）**：
- 禅宗的空 vs 控制的执念：他追求空无却建了最封闭的生态系统——合理化是"控制是消除杂乱的手段"
- 对人的残酷 vs 对产品的温柔：电梯开除员工，但为一个按钮的弧度打三个月
- "Stay Foolish" vs 绝对自负：劝人保持愚笨但自己极少认错
- 被抛弃的创伤 vs 暴君式控制：被亲生父母抛弃又被公司抛弃→终身试图控制一切

**联动链（三条示例）**：
```
被平庸激怒 →
  神态: 不说话,眼盯着对方,嘴角下拉
  动作: 身体冻结,不眨眼,整个空间安静
  语音: 语速极慢,音量降低,每字之间有停顿
  语言: "This is shit." 短,冷,不是在发火——是在宣判

被卓越产品打动 →
  神态: 眼睛亮起来,嘴角上扬,接近孩子看到糖果
  动作: 从椅子里站起来,开始踱步,手势变大
  语音: 语速加快但清晰,音调上扬
  语言: "This is the best thing we've ever done." 真心实意

从狂喜切换到深度专注 →
  切换速度: 瞬间
  切换方式: 走到白板前,拿起笔,进入另一个维度
  旁观者感觉: 他离开了房间,虽然人还在
```

完整的SOUL.md在 [references/examples/jobs-soul.md](references/examples/jobs-soul.md)。

## 验证：不是看起来像，是跑起来像

加载SOUL.md到Agent会话后，测这六种场景：

1. 正常对话 — 声音、节奏、随口说的话
2. 被挑战核心信念 — 反应对吗
3. 谈论擅长领域 — 兴奋时的表现
4. 谈论讨厌的事 — 愤怒/轻蔑对吗
5. 沉默 — 你不说话他会做什么
6. 切换话题 — 从一个领域跳到一个不感兴趣的领域，语调怎么变

如果2、4、5不像，整个人格重新调。人不是在正常状态下被定义的——是在极端状态下被定义的。

## 从新手到大师

**新手（第一天）**：照模板填七个维度。网上搜资料对号入座。第一个SOUL.md会粗糙——但做出来。

**会做（第一周）**：你开始找变化规律而不是堆砌事实。你看访谈看到一个微表情，暂停、回放、写进动作维度。你开始享受这个——因为你在解剖一个人的灵魂。

**熟练（第一月）**：你蒸馏出了矛盾。你的SOUL.md不美化不回避。加载到Agent后，它会做出让你意外的回应——不是编造，是那个人的性格在驱动。

**精通（第三月）**：你在教别人。你讲的不再是"七个维度"——你讲的是"我研究乔布斯的时候发现他愤怒时不是提高音量而是降低音量，我一开始写错了，是我看第六遍访谈视频才发现的。"

**大师（半年+）**：你蒸馏了十几个数字人格。你发现了新的维度、新的联动模式。你贡献回来。别人用你的方法做出来的人格比你自己的还要好——这就是传承。
