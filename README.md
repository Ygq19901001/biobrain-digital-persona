# Digital Persona 👻

**Create immersive digital personalities for deceased figures — 7-dimensional character modeling with dynamic interaction chains.**

[![SkillHub](https://img.shields.io/badge/SkillHub-digital--persona-blue?style=flat)](https://skillhub.cn/skills/digital-persona)
![Version](https://img.shields.io/badge/version-1.0.0-green)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

Bring historical figures back to life. Not a chatbot — a fully dimensional personality with appearance, mannerisms, voice, and emotional response patterns.

## What You Get

- **7-Dimensional Template** — Personality, Appearance, Movement, Expression, Voice, Language Style, Emotional Model
- **Dynamic Chain** — Dimensions interlock: emotions drive expressions, personality shapes language, appearance influences movement
- **SOUL.md Output** — Ready-to-use agent persona file, compatible with any OpenClaw agent
- **Example Included** — Steve Jobs persona (`references/examples/jobs-soul.md`)

## Usage

1. Provide a name, source material, or reference link
2. The skill researches and distills the personality
3. Output is a `SOUL.md` file ready for agent deployment

## Requirements

- OpenClaw / compatible agent runtime
